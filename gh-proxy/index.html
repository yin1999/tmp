GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持release、archive、文件以及网页浏览，右键复制出来的链接都是符合标准的，更多用法、clone加速请参考这篇文章。

release、archive使用cf加速，文件会跳转至JsDelivr

注意，不支持项目文件夹
